A modern introduction to learning Python using music and debating skills

By Lee-Anne Tan, Secondary 2 student, Methodist Girls' School

This beginner-friendly workshop blends rhythm and rhetoric to make Python intuitive and fun. Learn core concepts like variables, loops, functions, and problem decomposition through music patterns and structured debate techniques. Ideal for students, educators, and anyone keen to approach programming creatively.
